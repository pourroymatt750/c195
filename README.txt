C195 Software II Project: Appointment Scheduler

Purpose: The purpose of this Appointment Scheduler is to allow users to create customers, create
appointments for those customers, and to view reports on these two categories.

Author: Matthew Pourroy
Contact: pourroy.matt@gmail.com
Student application version: 1.0
Date: 1/8/2023
IDE: IntellJ Idea 2021.1.3 (Community Edition)
JDK: Amazon Correto 18.0.2
JavaFX version: JavaFX-SDK-18.0.2

How to run program: User must download MySQLWorkbench, connect to MySQL Connector driver, connect
to MySQL database, make sure correct JDK, SDK, and JavaFX version, and extract zip file to gain
access to program. To login, one an use Username: test, Password: test, to login and test the
application. 

Additional Report: The additional report provided in the application is a report that shows
the number of customers in each country. It is shown in a tableview with two columns, "Country"
and "Total Customers".

MySQL Connector driver version number: mysql-connector-java-8.0.25
